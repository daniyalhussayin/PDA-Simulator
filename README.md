# PDA-Simulator
 A Pushdown Automaton (PDA) Simulator built using Python and Streamlit. 
This project allows users to simulate and visualize how a PDA processes strings for specific formal languages.

## Features
- Simulates **L = { aⁿ bⁿ | n ≥ 1 }**
- Simulates **L = { w c wᴿ | w ∈ {a, b}* }**
- Step-by-step PDA execution
- Stack visualization
- Acceptance / Rejection result
- Clean table format using Streamlit
